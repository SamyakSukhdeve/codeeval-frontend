# Markdown Syntax Guide

## Headings

# This is an H1

## This is an H2

### This is an H3

---

## Text Formatting

- **Bold**: `**text**` → **text**
- _Italic_: `*text*` → _text_
- ~~Strikethrough~~: `~~text~~` → ~~text~~

---

## Lists

### Unordered List

- Item 1
  - Subitem 1.1
  - Subitem 1.2
- Item 2

### Ordered List

1. First item
2. Second item
3. Third item
